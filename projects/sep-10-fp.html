SEP10 Freedom Project

This project involved me researching a certain topic (I chose oceanography) and software/hardware involved in that topic.

I used SASS and HAML to make my website, which are preprocessors for CSS and HTMl respectively.

Throughout this project I honed many different skills including ones not just limited to SEP, such as time management and problem solving.

I had many takeways from doing this project, such as learning that it is absolutely necessary to read the documentation for tools in order to avoid mistakes. I also learned that it is important to read the error logs when using a compiled language, as much of the time when coding by reading the error logs I was able to avoid making the same mistakes. The next time I create a project like this, I will apply these takeaways that I learned so that I can become more productive.
